## 📂 Projects

### 🔐 Fraud Detection Pipeline

A production-oriented machine learning system designed for **high-risk financial environments**, where mistakes have real consequences.

#### ⚙️ What Was Built

* End-to-end ML pipeline (data → decision)
* Logistic Regression & Random Forest models
* SMOTE for imbalance correction
* RandomizedSearchCV for tuning
* Threshold optimization (0.5 → 0.3)

---

#### 📊 Key Outcomes

* Fraud Detection Recall: **95%**
* Precision: **63%**
* ROC-AUC: **~0.98**
* PR-AUC: **0.88**
* False Positives reduced from **730 → 11**

---

#### 💡 Why This Project Stands Out

Most models:

* Look good on metrics
* Fail in real-world usage

This system:

* Captures **maximum fraud cases**
* Keeps false alarms **under control**
* Balances **risk vs user experience**

👉 Built with a focus on **decision quality, not just prediction accuracy**
